# 撰寫方法論紀錄 — 主題候選三

以參數化 Workflow（`topic-review.js`）跑 scout→fetch→deepdive→fetch2→draft→audit。

## 產出量
- **Scout**（5 維度：confidence_trial / hyperkalemia_combo / combination_editorial / guideline_position / glp1_cardiorenal）：去重後 **40 篇候選**。
- **Fetch**：**22 篇** 全文；**Fetch2**：再 5 篇。取得 CONFIDENCE 完整家族（main abstract、baseline NDT、eGFR mediation JASN、BP、JACC hyperkalemia、sexual dimorphism）+ GLP-1 cardiorenal canon（Perkovic FLOW 2024、Mann 2024、Kristensen 2019、Neuen 2024、Pratley）+ guideline（KDIGO 2024/2022 PDF、ADA ElSayed 2025）。
- **Draft**：正文約 6,500–7,000 中文字,203 行;6 個 H2;背景 ~30% / 深度 ~70%;2 個 mermaid（CONFIDENCE 三臂 schema、四支柱決策流程）+ 四支柱作用層次表 + UACR-vs-eGFR-dip 圖 + benefit-vs-hyperkalemia 平衡圖。
- **引用**：25 篇 Vancouver + DOI;19 個不重複本地檔 tag,全部 grep 可稽核。
- 三大爭議以 Georgianos+Cheng 為正方、JACC+Sinclair+Wen 為保留方;明確區分 surrogate-based expansion / hypothesis-generating / 模型外推 / guideline routine。
