# 風險基礎 HTE

> 以多變量風險模型取代單一變量 subgroup：先估個人基線事件率，再乘上穩定的相對效果。

## 定義與定位

風險基礎的治療效果異質性（risk-based heterogeneity of treatment effect, HTE）主張把次族群分析從「切變數」改成「估風險」。Kent 2016 分析 32 個大型試驗、39 個分析後發現：18 個整體顯著的比較中只有 1 個在比例尺度上有顯著的「治療×基線風險」交互，但極端風險四分位間的 ARD 差中位 5.1%、四分之一的試驗超過 10%；極端四分位風險比（EQRR）中位 4.3、最高達 50.7，且多數病人的風險低於試驗平均值。Kent 2018 的 PATH statement 據此主張以 reference class forecasting／多變量風險模型取代單一變量 subgroup。對應到 finerenone，就是以 eGFR×UACR（必要時加上 HF 病史、LVH）估個人基線事件率，再乘上穩定的相對效果。
