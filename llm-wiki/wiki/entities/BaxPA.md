# BaxPA

> [[Baxdrostat]] 在原發性醛固酮症的專屬 Phase 3 試驗，**進行中**；本知識庫只記錄到它的存在與期別，未收錄其設計與結果細節。

## 定義與定位

BaxPA 是 baxdrostat 針對確診 [[原發性醛固酮症]] 的 Phase 3 試驗。本知識庫目前僅記錄到此試驗**存在、期別為 Phase 3、正在進行**，語料明白寫道「BaxPA 僅知其存在與期別」——其樣本數、主要終點、滴定規則與讀出時間均未收錄。它在本知識庫中的意義不在既有結果，而在於它是**英文投稿稿提出的可證偽要求的收件人**之一。

證據層級上，[[Baxdrostat]] 目前在 PA 的一手證據僅有 [[SPARK]]（N=15、開放單臂），BaxPA 是把該類別推向 PA 專屬 Phase 3 的關鍵；但 ASI 類別至今仍**沒有任何 hard-outcome 讀出**，[[EASi-KIDNEY]] 才是首個 ASI hard cardiorenal outcome phase 3 且仍進行中。**尚待驗證**

## 不同素材中的觀點

### 來自 [[2026-07-22-07-submission_EN_manuscript_clean]]

英文投稿稿把 BaxPA 當成一個具體交付物的對象：**BaxPA 與 [[FAIRY]] 必須在每一個 renin 時間點同步測 [[11-DOC]]，並預先訂好「renin 上升但 11-DOC 升高」時的判讀規則**，否則兩者的主要生化終點在跨類別上不可比。稿件形容這個要求「便宜且可證偽」——若 11-DOC 被測了卻與 renin 軌跡無關，該 exhibit 即被推翻，「the field will have lost nothing but an assay」。同稿在對照其 Table 3 規格書檢視現行試驗時判定：FAIRY 方向正確但踩了兩個坑，**BaxPA 僅知其存在與期別**，RETAME-PA 是唯一以 renin 滴定者但屬 feasibility。<!-- confidence: EXTRACTED -->

### 來自 [[2026-07-22-07-07_aldosterone_synthase_inhibitor_vs_finerenone]]

在 SPARK 的證據層級評註中並列指出：SPARK 為 N=15、開放單臂、無 MRA/finerenone 對照、12 週主要窗，屬假說生成；「Phase 3 BaxPA 進行中」。並在定位論述中重述 baxdrostat 雖已有 SPARK 的確診 PA 一手數據，其 **PA 專屬 Phase 3（BaxPA）仍進行中**，故 ASI vs MRA 在 PA 的 head-to-head 尚不存在。
