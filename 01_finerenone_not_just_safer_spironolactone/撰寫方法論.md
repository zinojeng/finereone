# 撰寫方法論紀錄 — 主題候選一

套用 `medical-literature-review` skill v2.0 的「三層外部知識管線 + 第二輪深挖 + 敵對稽核」，以 Workflow 多 agent 編排執行。

## 管線與產出量
- **Phase Scout**（5 維度並行 literature-scout）：molecular / preclinical-antifibrotic / ARTS-program / biomarker / HF-FINEARTS。去重後 **27 篇候選**，每篇經 PMID/DOI 驗證真實存在。
- **Phase Fetch**（fulltext-fetcher，download_with_fallback → 本地 MD，肉眼驗證防 Polluted PDF）：**14 篇** 取得全文。
- **Phase DeepDive + Fetch2**（deep-dive-scout 抽高頻共引 seminal papers）：第二輪再取 **3 篇** 全文（如 Amazit 2015、Geller 2000、Fagart 2010、Coresh 2019 等 MR 分子/UACR-surrogate 原始研究）。
- **Phase Draft**（drafter，只引用本地 MD）：正文約 19,091 非空白字元，10 個 H2 段落，涵蓋引言→臨床背景→三層（分子/生物標記/臨床）→爭議對讀→未來照顧建議；含表 1（12 列藥理學差異）、Mermaid 流程圖 1、圖 2 轉譯路徑文字、22 篇 Vancouver References。
- **Phase Audit**（citation-verifier / claim-auditor / cross-referencer 並行 + drafter 修正回合）：共 8 條 findings，其中 **3 條必修全部處置完成**。

## 攔下 / 修正的問題
1. **META_MISMATCH（Ref #5 ARTS）**：References 缺 DOI → 經 Crossref/WebFetch 確認後補 `10.1093/eurheartj/eht187`。
2. **META_MISMATCH（Ref #20 FIND-CKD）**：出版年 2025 → 更正為 2026（對齊 Crossref published_date 2026-06-04）。
3. **grep-miss dead-link（第 154 行）**：死鏈檔名標註 → 改為已列於 References 的 FIDELIO/FIGARO（該劑量/暫停方案出自二試驗 Methods）。

## 資源用量
44 個 agent（43 完成、1 因內容過濾失敗：ARTS-HF fetch）、617 tool calls、subagent tokens ~2.29M、耗時 ~38 分。

## 硬規則落實
LLM 僅做組織與改寫；所有數字先 Read/Grep 本地 MD 取得（FIDELITY UACR 相對降 32%、CV HR 0.86、腎 HR 0.77、ARTS 血清鉀 0.04–0.30 vs 0.45 mmol/L、Amazit IC50/Kd/S810L、Coresh HR 0.83/0.78、FINEARTS rate ratio 0.84、FIND-CKD eGFR 斜率差 0.7 等），逐一附檔名供 grep 稽核。
